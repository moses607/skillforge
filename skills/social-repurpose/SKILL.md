# Social Repurpose

One asset, many native posts. The mistake is copy-pasting the same blurb everywhere. Each platform rewards a different shape — respect it. Extract the *ideas* from the source, then re-form each for its channel.

## Step 1: mine the source

Pull out, before writing any post:
- The **one big idea** (the thing worth spreading).
- 3-5 **standalone insights** (each true and useful on its own, no context needed).
- 1-2 **surprising data points or contrarian claims** (these travel).
- A concrete **example or story** if present.

If the source has no idea worth spreading, say so — don't manufacture engagement bait.

## Step 2: shape per platform

**X / Twitter (single)** — one insight, one hook line, ≤280 chars, no hashtags-as-filler. Front-load the payoff.

**X / Twitter (thread)** — hook tweet that promises a specific payoff → one insight per tweet → last tweet = recap + soft CTA. 5-9 tweets. Each tweet must stand alone if screenshotted.

**LinkedIn** — hook line, then a line break (the "see more" cutoff lives here — the first line must earn the expand). Short paragraphs, 1 idea each, a lesson or POV, light CTA/question. 1200-1600 chars. No engagement-bait ("Agree?" spam).

**Instagram / carousel** — slide 1 = hook, slides 2-N = one point each (≤15 words/slide), final slide = CTA. Give slide-by-slide text.

**Newsletter blurb** — 2-3 sentences + link, written to make opening the full piece feel worth it.

## Output template

```
## Repurposed: <source title>

**Big idea:** <one line>

### X — single
<post>

### X — thread
1/ <hook>
2/ ...

### LinkedIn
<post, with the first line isolated>

### Instagram carousel
Slide 1: <hook>
Slide 2: ...

### Newsletter blurb
<2-3 sentences + [link]>
```

## Rules

- Never reuse the identical sentence across platforms — re-shape it.
- Match the source's actual claims; don't inflate a modest finding into a bombshell.
- No hashtag stuffing, no "🚨 THREAD 🧵" theatrics unless the user's brand does that.
- If the user names only one platform, produce only that — deeply, with 2 variants to choose from.
